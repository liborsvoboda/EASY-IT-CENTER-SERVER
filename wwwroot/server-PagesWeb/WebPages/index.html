Celé IT = SERVERY, SYSTÉMY, WEBY

SERVERY

Servery jsou pro poskytování služeb. A přesně tak je uchopen projekt EIC EasyITcenter. Server má v sobě naimplemetován více jak 300 rozšíření OPEN SOURCE. A nejen to, programově je kompletně vyladěn pro centralizované použití = tzn. každá služba funkce, definice je pečlivě sestrojena 1x nebo je využíváno generické nastavení. To znamená že služba definuje nezávisle samu sebe, aniž by muselo být v programu rozepsáno jak vypadá. A toto celé se dá vytvořit připojeným ONLINEM Editorem, který uloží nastavení do Databáze. Služby lze tedy vytvářet Dynamicky z Editoru a Server je automaticky ihned po uložení Publikuje. Toto hravě zvládne Každý IT Administrátor. a Takto lze pokrýt Celé IT jedniným Serverem.

SYSTÉMY

Projekt ESB EasySYSTEMbuilder je právě takové řešení. Dynamický SYSTÉM s již vytvořenými více jak 100 Agendami a s implementovaným Designerem pro přidávání nových AGEND pouze jednoduchým vytvořením Databázové Tabulky. K ni se automaticky po zařazení tabulky do Systémového menu a nastavení Práv vygeneruje Formulář a nová Agenda je na Světe. Systém podporuje pomocí dalších 300 implemetovaných Rozšíření naprosto cokoliv - Dotykové panely, multimediální Systémy, 3D, Videa, Audia, Dokumenty, Editory, více jak 500 nachystáných nástrojů pro Editace a tvorbu v podobě Webových mikroaplikací. Aplikace je otevřena v Agendě jako editor. A s pouhým nastavením Input/Output takto můžete přidávat vlastní Nástroje bez Omezení. Navíc, můžete využít sdílených webů a mikroaplikací Serveru jako Systémové Agendy.

WEBY

Řešení má v sobě nachystáno již více jak 50 Editorů, 20 Generátorů, a GIGA Web knihovnu Statických nástrojů a Frameworků o velikosti 60GB. Můžete pouřít i Implemetované Generátory Webových Aplikací, či Online generátory kódu a již Brzy i Web Online Editor s grafickými Studii atd... Server je Vlastně Kompletní HOSTING. můžete mít neomezený počet Webů portálů, či cokoliv si zamanete. a to vše pouhým zkopírováním již připravených řešení v GIGA knihovně. Napojení na API můžete nasdílet se Systémeme a v Systému tak zobrazit libovolný WEB jako Agendu, která je přístupná jak ze SYSTÉMU, tak i z WEBU. Takže si sami můžete zvolit jakou cestou a jakým způsobem si chcete Řešení Přizpůsobit vlastním potřebám.
